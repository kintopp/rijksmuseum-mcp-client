# rijksmuseum-mcp-client

Web-based [MCP](https://modelcontextprotocol.io) client for the [Rijksmuseum MCP+](https://github.com/kintopp/rijksmuseum-mcp-plus) server.

**Split-pane UI:** OpenSeadragon IIIF viewer (left) + chat interface (right). Bring your own LLM via [OpenRouter](https://openrouter.ai) — pick any tool-use-capable model from a dropdown and start exploring the Rijksmuseum collection.

## Status

Early development. See [kintopp/rijksmuseum-mcp-plus-offline#190](https://github.com/kintopp/rijksmuseum-mcp-plus-offline/issues/190) for the design discussion.
