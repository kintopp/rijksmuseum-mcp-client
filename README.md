# rijksmuseum-mcp-client

Web-based [MCP](https://modelcontextprotocol.io) client with [MCP Apps](https://modelcontextprotocol.io/extensions/apps/overview) support for the [Rijksmuseum MCP+](https://github.com/kintopp/rijksmuseum-mcp-plus) server.

**Split-pane UI:** OpenSeadragon IIIF viewer (left) + chat interface (right).
